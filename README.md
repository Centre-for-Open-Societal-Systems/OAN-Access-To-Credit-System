# Open Agri Access Credit System

This project recreates the attached login/role-selection screen in React using Vite, Tailwind CSS, and SCSS.

## Stack

- React 18
- Vite
- Tailwind CSS
- Sass / SCSS
- lucide-react icons

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- The page uses SCSS for the theme, layout polish, and shared component styles.
- Color tokens are exposed through CSS variables so the Tailwind theme and SCSS stay aligned.
- The role cards are interactive and default to the Development Agent state shown in the design.
